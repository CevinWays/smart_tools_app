# Contributing to SiagaWarga

Terima kasih atas minat Anda untuk berkontribusi pada **SiagaWarga**! Kami sangat menghargai bantuan Anda untuk membuat aplikasi ini lebih baik bagi masyarakat.

## Cara Berkontribusi

### 1. Melaporkan Bug (Issues)
Jika Anda menemukan bug atau masalah, silakan buat **Issue** baru dengan detail berikut:
- Deskripsi masalah.
- Langkah-langkah untuk mereproduksi masalah.
- Screenshot (jika ada).
- Informasi perangkat (OS, versi Flutter).

### 2. Mengajukan Fitur Baru
Punya ide fitur keren? Buat **Issue** dengan label `enhancement` dan jelaskan ide Anda serta bagaimana fitur tersebut akan membantu pengguna.

### 3. Pull Requests (PR)
Kami menerima kontribusi kode! Ikuti langkah ini:
1.  **Fork** repository ini.
2.  Buat branch baru untuk fitur/fix Anda: `git checkout -b fitur-keren-anda`.
3.  Commit perubahan Anda dengan pesan yang jelas: `git commit -m "Menambahkan fitur X"`.
4.  Push ke branch Anda: `git push origin fitur-keren-anda`.
5.  Buat **Pull Request** ke branch `main` repository ini.

## Panduan Coding
- Gunakan **Dart Analyzer** dan **Flutter Lints** untuk memastikan kode bersih.
- Ikuti struktur folder yang ada (`lib/features/`, `lib/core/`).
- Gunakan **BLoC/Cubit** untuk state management.
- Pastikan tidak ada error sebelum submit PR.

## Lisensi
Dengan berkontribusi, Anda setuju bahwa kontribusi Anda akan dilisensikan di bawah **MIT License**.
